# notion-md-pdf

Converts [Notion](https://notion.so)-exported Markdown files into PDFs

## Installation

Yarn

```bash
yarn global add notion-md-pdf
```

NPM

```bash
npm install -g notion-md-pdf
```

## Usage

```bash
notion-md-pdf ~/Export-XXXXXX.zip
# Conversion completed: ~/Document-Name-XXXXXX-1.pdf
```
